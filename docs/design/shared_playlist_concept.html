RECOMMENDED FEATURE

음악 노트

각자 노래 링크와 짧은 이유를 남기면 둘만 볼 수 있는 플레이리스트에 쌓입니다. 실제 음악 재생 API 연동은 MVP에서 제외하고, YouTube Music, Spotify, Apple Music 링크를 붙여 넣는 방식으로 시작합니다.

MVP RULES
  • 곡 제목, 아티스트, 링크, 80자 메모, mood chip만 저장합니다.
  • 반응은 `들어봤어요`, `나중에 들을게요`처럼 낮은 압력의 상태만 둡니다.
  • 하트, 커플, 사랑 같은 표현은 사용하지 않습니다.
