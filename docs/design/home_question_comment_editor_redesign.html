TODAY'S QUESTION 댓글 수정 UI 개선안

댓글은 대화 스레드가 아니라 상대 답변 아래에 붙는 짧은 코멘트입니다. 그래서 수정 UI도 큰 채팅창처럼 보이기보다, 같은 카드 안에서 읽기와 편집 상태가 안정적으로 전환되는 구조가 좋습니다.

느슨한 링크 제거

`댓글 수정하기`를 본문 아래 텍스트 링크로 띄우지 않고, 댓글 카드 오른쪽의 작은 편집 아이콘으로 정리합니다.

편집은 같은 자리에서

수정 버튼을 누르면 기존 댓글 카드가 편집 패널로 확장되어 맥락을 잃지 않습니다.

저장 부담 낮추기

120자 제한, 취소, 수정 저장, 오류 문구를 한 패널 안에 묶어 작고 명확한 작업처럼 보이게 합니다.

기본 상태

댓글을 카드로 분리

내 댓글을 상대 답변 하위 card로 보여주고, 수정은 우측 아이콘으로만 노출합니다.
