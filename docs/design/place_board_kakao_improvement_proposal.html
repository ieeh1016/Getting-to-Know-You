KAKAO PLACE BOARD REDESIGN

장소 기능을 “지도 붙은 게시판”에서
“같이 갈 곳을 고르는 흐름”으로 바꾸기

현재 장소 화면은 카카오 검색, 지도, 저장 카드가 각각 따로 보입니다. 개선안은 검색 결과를 고르면 지도와 저장 카드가 즉시 연결되고, 저장된 장소 카드에서 바로 다음 행동을 할 수 있게 만드는 방향입니다.

지금 아쉬운 점

  • 상단 지도 안 검색처럼 보이는 영역과 실제 검색 폼이 중복됩니다.
  • 검색 결과 선택 후 지도에 반영되는 감각이 약합니다.
  • 저장된 장소 카드에서 할 수 있는 행동이 부족합니다.
  • 검색 실패 원인이 도메인 등록 문제인지 알기 어렵습니다.

개선 원칙

  • 검색은 한 곳에서만 시작하게 합니다.
  • 결과 선택 즉시 지도 중심, 마커, 저장 미리보기가 같이 바뀝니다.
  • 장소 카드는 지도 열기, 관심 표시, 메모 수정을 바로 제공합니다.
  • 카카오만 사용한다는 점을 UI와 데이터 모델에서 일관되게 보여줍니다.
