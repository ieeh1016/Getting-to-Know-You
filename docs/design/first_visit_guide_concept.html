동작 기준

첫 방문 안내는 앱 데이터가 준비된 뒤 홈에서 노출하고, 사용자가 둘러보기나 바로 시작하기를 누르면 같은 기기에서는 다시 자동으로 띄우지 않습니다.

저장 키는 jogeumssik:onboardingSeen:{spaceId}:{profileId} 형태를 권장합니다.
Firestore에는 안내 읽음 상태를 저장하지 않아 무료 플랜 비용을 늘리지 않습니다.

화면 배치

홈에는 첫 방문 안내만 잠깐 노출하고, 전체 기능 설명은 마이 > 도움말 > 처음 안내 다시 보기로 이동합니다. 홈의 밀도는 유지하고 처음 사용자의 불안감만 줄입니다.

로맨틱한 표현 대신 기능 중심의 담백한 문구를 사용합니다.
둘러보기는 단계형 화면으로 처리해 사용자가 현재 위치를 바로 알 수 있게 합니다.
