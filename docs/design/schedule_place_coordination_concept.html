PLUS FEATURE PROPOSAL

만날 날과 가고 싶은 곳을
조용히 맞춰보는 두 가지 기능

기존 `조금씩`의 무압박 톤을 유지하면서, 질문과 음악 이후 자연스럽게 다음 만남으로 이어지는 기능을 제안합니다. 핵심은 확인, 공유, 기록이고 실시간 추적이나 강한 알림은 배제합니다.

01 · 일정 조율

서로의 여유가 겹치는 날을 찾는 캘린더

각자 가능한 날짜와 시간대를 가볍게 체크하면 둘 다 가능한 날이 자동으로 강조됩니다. 일정 제목을 자세히 공개하지 않아도 `가능`, `어려움`, `조율 필요` 정도만 공유할 수 있어 부담이 낮습니다.

  • 월간 캘린더에서 서로 가능일, 내 가능일, 상대 가능일을 색으로 구분
  • 날짜를 누르면 오전, 오후, 저녁 단위로 가능 여부와 짧은 메모 입력
  • 둘 다 가능해진 후보는 `살짝 잡아두기`로 약속 후보 상태 저장
02 · 장소 보드

지도에서 가보고 싶은 곳을 함께 모으는 기록

카카오 지도 API를 통해 장소 검색, 지도 표시, 마커 저장을 제공합니다. 내 위치를 계속 공유하는 기능이 아니라, 둘이 선택한 장소만 보드에 남기는 방식입니다.

  • 지도 위 마커와 하단 장소 목록으로 한눈에 탐색
  • 장소별로 `내 관심`, `상대 관심`, `서로 관심` 상태와 짧은 메모 기록
  • 일정 조율 기능과 연결해 `이 날짜에 갈 곳 후보`로 묶기
