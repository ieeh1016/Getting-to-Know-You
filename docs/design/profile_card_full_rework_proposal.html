구현 방향

  • 1현재 `ProfileSlot` 기능은 유지하고, 화면을 `요약 히어로 → 직접 추가 → 작성 카드 → 편한 질문 → 숨긴 질문` 순서로 재배치한다.
  • 2기본 질문은 그리드 전체 노출보다 “추천 2개 + 카테고리 필터”로 줄여 숙제 느낌을 낮춘다.
  • 3직접 만든 카드는 작성 카드 목록 상단에 우선 노출하고, `직접 카드` 배지를 붙인다.
  • 4상대 카드에서는 읽기 전용 피드로 보여주며, 새로 추가된 카드만 상단에 별도 섹션으로 올린다.

정책 유지

  • A직접 카드 + 답변 있음: 상대 카드에 표시한다.
  • B직접 카드 작성 중 또는 답변 없음: 상대 카드에 표시하지 않는다.
  • C기본 질문 숨기기: 내 화면에서 숨기고 상대에게도 보이지 않게 한다. 복구 가능하다.
  • D직접 카드 삭제: Firestore 문서 삭제로 처리하고 상대 화면에서도 사라지게 한다.
